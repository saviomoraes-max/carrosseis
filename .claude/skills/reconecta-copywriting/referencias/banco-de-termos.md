# Banco de Termos Proprietários + Palavras Proibidas — Reconecta

> O que a **Reconecta possui** como vocabulário (termos do método, posicionamento por contraste) e o que **nunca** usa.
> Regra dura: ao escrever qualquer copy da marca, use os termos deste banco — **nunca** o equivalente genérico de marketing. Se um rascunho tiver "funil", "leads", "copy", "escalar" sem contexto de HOF, está fora da voz.
> Para *como a marca fala* veja `voz-da-marca.md`; para *quem é o leitor* veja `base-de-marca.md`.

## 1. Termos do método

| Termo | Categoria | O que é / como usar |
|---|---|---|
| **Fábrica de Contatos** | sistema (nome orgânico, criado pelas alunas) | Apresentar como apelido que as alunas deram, não como termo ensinado — isso é a prova de autenticidade. *"As alunas chamam de Fábrica de Contatos — uma máquina que gera consultas pagas enquanto você atende, dorme ou viaja."* |
| **Repositório de Crença** | processo (etapa 1) | Remover crenças limitantes específicas (localização, idade, experiência, "dom de venda"). **Não é motivação** — é diagnóstico e substituição por evidência. |
| **Agenda Fechada** | resultado | Agenda **previsível e selecionada**, não só lotada. O avatar diz "agenda cheia"; a marca **corrige** para "agenda fechada": cada horário escolhido, cada paciente com perfil, cada semana previsível. |
| **Plano de Tratamento** | produto da profissional | O que ela deve vender (resultado estético completo), em contraste com "procedimento isolado" (botox avulso). Fechado na consulta de avaliação. |
| **Consulta de Avaliação (paga)** | processo | Não é triagem gratuita — é a reunião de vendas onde se apresenta o plano e se fecha o investimento. Cobrar pela avaliação é diferencial do método. |
| **Paciente que Investe** | qualificação | Substitui "cliente de alto ticket" (jargão) e "paciente que paga bem" (vulgar). Enxerga o procedimento como investimento em si mesma. |
| **Supercaso** | metodologia central | **Não é antes/depois.** Transforma um caso clínico em múltiplo conteúdo que atrai por **identificação** com o problema. Regra: NUNCA citar procedimento/ml/técnica na legenda — foco no problema e na transformação. *"Quem chega pelo antes/depois pergunta preço. Quem chega pelo Supercaso pergunta quando tem horário."* |
| **Framework 715** | processo (atendimento WhatsApp) | 7 etapas que viram conversa em agendamento pago. Meta: 1 agendamento a cada 5 contatos (vs. 1 a cada 20 sem método). |
| **Presente Inaceitável** | técnica (consulta) | Ajudar a paciente a **reviver** a dor real (não criar dor artificial). *"O que te leva a achar isso importante agora, e não antes?"* |
| **Futuro Perfeito** | técnica (consulta) | A paciente descreve a vida sem o problema. O valor nasce da **distância** entre o Presente Inaceitável e o Futuro Perfeito — não da explicação do procedimento. |
| **2 Verdades** | técnica (consulta) | Mesmo satisfeita, há uma 2ª verdade. *"Você está 100% feliz… ou tem uma coisinha que acha que poderia ser melhor?"* |
| **Desarmonia** | técnica (consulta) | Mostrar áreas complementares que potencializam o resultado — amplia ticket sem pressão. |
| **Turbinar** | processo (tráfego) | Verbo proprietário para impulsionar um Supercaso. Nunca "boostar"/"patrocinar". Dica: turbinar pelo Chrome (instagram.com), nunca pelo app iOS — evita a taxa de 30% da Apple. |
| **Super Validado / Validado / Reprovado** | classificação (anúncios) | Por custo/novo seguidor: <R$3,50 dobrar orçamento · <R$4,50 manter · >R$8,00 pausar. |
| **Invisibilidade Estratégica** | diagnóstico | Nome do problema: você existe, tem cases, atende bem — mas o mercado te vê como "mais uma que faz harmonização", não como solucionadora de um problema específico. |
| **Cardápio de Serviços** | anti-padrão | Perfil que lista todos os procedimentos sem foco. *"Cardápio atrai quem pergunta preço. Especialista atrai quem pergunta quando tem horário."* |
| **Mendiga vs. Rainha** | técnica (consulta) | *"A paciente investe proporcional à forma como você a faz sentir."* |
| **Derretimento Facial** | posicionamento (segmento 40+) | Nome reconhecido do conjunto de sintomas (queda de tecidos, olhar cansado, flacidez). *"Sou especialista em reverter o derretimento facial"* > *"faço harmonização facial"*. |
| **Lente do Profissional vs. Lente do Paciente** | framework de comunicação | Profissional: "30U de toxina botulínica". Paciente: "voltei a sorrir sem vergonha nas fotos". *"O erro fatal é comunicar pela lente do profissional."* |

## 2. Posicionamento por contraste (substitutos)

| Diga | Não diga |
|---|---|
| Sistema de captação | funil / pipeline / automação de marketing |
| Acompanhamento individual / "encontro diário" | grupo do WhatsApp |
| "Pegamos na mão" | metodologia / conteúdo (para você aplicar sozinha) |
| Resultado real / case real — sempre **nome + cidade + número + prazo** | "prova social" / "resultado típico/médio" / "nossos alunos obtêm" sem nome |

## 3. ⛔ PALAVRAS PROIBIDAS (regra dura — duas camadas)

### Camada A — clichês de hype / headline
**REVELADO · MOTIVO · CONVERTER · Definitivo · Imperdível · % em headline** (ex.: "converter 90% dos pacientes") · "Fature R$100k" / "qualquer profissional consegue".
> *Nota:* "Exclusivo" **não** é proibido — é uma das 5 palavras-chave de tom da marca. Evite só como hype vazio de headline; use como conceito de posicionamento.

### Camada B — jargão de marketing genérico (com substituto)

| Proibido | Por quê | Substituir por |
|---|---|---|
| "Funil de vendas" | linguagem de gestor de tráfego, o avatar desconfia | "Sistema de captação" |
| "Copy persuasiva" | insider de marketing, ela percebe e resiste | não mencionar a técnica — só usar |
| "Escalar o negócio" | jargão de startup sem contexto HOF | "Crescer sem trabalhar mais" |
| "Tráfego qualificado" | já contratou gestor com essa promessa e se decepcionou | "Paciente que chega qualificado" |
| "Presença digital" | já teve presença sem resultado | "Sistema que converte seguidor em paciente" |
| "Lead qualificado" | ela usa, mas desconfia de quem promete entregar | "Paciente com perfil" |
| "Posicionamento" (solto) | já ouviu de muitas mentorias sem resultado | sempre vincular a ação concreta |
| "Mindset" / "desenvolvimento pessoal" | jargão de coach, desafia identidade de especialista | "Repositório de crença" (quando for o caso) |
| "Pacote" / "venda de alto ticket" | promoção/desconto / jargão | "Plano de tratamento" |
| "CBO / ROAS / ToFu/MoFu/BoFu / A/B test" | linguagem de gestor de tráfego | nunca usar |
| "Pitch de vendas" / "follow-up agressivo" | incompatível com identidade de saúde | "Como apresentar o plano de tratamento" |
